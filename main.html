Navigation Bar Example

This example demonstrates a navigation bar created using HTML lists and CSS display properties.

  • Home
  • About
  • Contact Us
  • Itinerary

Navigation Bar Features

This navigation bar meets all the specified requirements:

  • Background color: #D3D3D3 (light gray)
  • Padding: 10px
  • List items: Home, About, Contact Us, Itinerary
  • Displayed horizontally using display: inline
  • Margin-right: 1rem between items
  • No bullets using list-style: none
/* CSS for the navigation bar */
.nav-bar {
    background-color: #D3D3D3;
    padding: 10px;
}

.nav-list {
    list-style: none;
}

.nav-item {
    display: inline;
    margin-right: 1rem;
}
